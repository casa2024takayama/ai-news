スマホの Discord から自宅 Mac の AI を操作する

更新: 2026-05-27

概要 振り分け コマンド例 ツール 注意 ヘルプ

概要

1スマホ Discord で bot にメンション / DM
2自宅 Mac の Hermes gateway が受信
3内容に応じて Hermes または Cursor Agent が処理
4結果が Discord に返る（テキスト・画像・動画）

前提: Mac が起動中・gateway 常駐・スリープしていないこと。

振り分けルール（最重要）

やりたいこと誰に頼むDiscord の書き方
X 検索・トレンド Hermes x_search で …
画像・動画生成 Hermes image_generate … / video_generate …
Web 検索・天気 Hermes 普通に質問
cron・定期ジョブ Hermes cron …
コード修正・レビュー Cursor cursorに: …
リポジトリ調査 Cursor Cursorで …

⚠️ 「cursorに」と書かないと Hermes（Grok）が自分で答えようとすることがあります。

Discord コマンド例

Hermes 向け

x_search で "セゾンテクノロジー" を直近24時間検索して要約
image_generate で「白い猫、ミニマル」を生成して画像添付
video_generate で「海の波5秒」を生成して添付
execute_code で 2+2 を計算

Cursor 向け

cursorに: 直近コミットをレビューして
cursorに: kakebo の README を1段落で要約
Cursorで README の typo を直して

セッション操作

/new

ツール設定を変えたあとは /new で新セッション。古いスレッドには新ツールが載りません。

主要ツール名

設定名と実際のツール名が異なる場合があります。

機能ツール名
X 検索x_search
画像生成image_generate
動画生成video_generate
Web 検索web_search
コード実行execute_code
読み上げtext_to_speech
利用可能ツールの確認
利用可能なツール名を列挙して。video_generate と image_generate が含まれるか確認して。

注意・トラブル

  • ✓ Mac がスリープ中は Discord から無反応
  • ✓ 設定変更後は hermes gateway restart → Discord で /new
  • ⚠ 動画生成は 1〜4 分・クォータ消費大
  • ⚠ Cursor は --force でファイルを書き換える可能性あり
